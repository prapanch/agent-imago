# Agent Imago

## How to use these skills

This repository contains skills organized as individual SKILL.md files. Each skill is self-contained.

**For OpenClaw agents:** Install via `openclaw skill install <path>`
**For Claude Code agents:** Use `CLAUDE.md` in this repo as context
**For Gemini agents:** Use `GEMINI.md` in this repo as context
**For other agents:** Read individual SKILL.md files directly — each is plain markdown
